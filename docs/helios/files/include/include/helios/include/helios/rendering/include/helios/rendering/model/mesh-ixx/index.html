45 * This variable is used to track whether the mesh's derived properties, such
46 * as its axis-aligned bounding box (AABB), are outdated and need to be recalculated.
47 * It is marked as mutable to allow modification within const member functions.

52 /**
53 * @brief Represents the axis-aligned bounding box (AABB) for the mesh.
54 *
55 * This member variable stores the computed or cached AABB for the mesh in
56 * 3D space. The AABB is a minimal bounding box aligned with the coordinate
57 * axes that encloses all the vertices of the mesh. It is marked mutable to
58 * allow updates during read-only operations, such as recalculating bounds
59 * as vertex positions are modified or updated dynamically.
60 *
61 * @see helios::math::aabb
