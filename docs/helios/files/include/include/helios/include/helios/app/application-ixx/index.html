26 /**
27 * @brief Class providing the entry point for a helios application.

29 * `Application` serves as the central hub for the helios framework, offering convenient access
30 * and management for key subsystems such as the EventManager, the InputManager and the
31 * RenderingDevice.
32 *
33 * It also allows for the registration of `helios.app.controller.Controller`s,
