40 /**
41 * @brief Represents a rectangular area within a RenderTarget where a scene is rendered.
42 *
43 * A Viewport defines the 2D rectangle into which a 3D scene is projected. Its dimensions are specified
44 * in normalized coordinates relative to its parent `RenderTarget`. It is associated with a `CameraSceneNode`,
45 * which provides the view and projection matrices. The viewport automatically updates the CameraSceneNode
46 * associated camera's aspect ratio when its parent `RenderTarget` is resized.
47 *
48 * Since a Viewport only stores normalized values for its location and dimensions relative to the owning RenderTarget,
49 * rendering APIs should query the size of the RenderTarget when a viewport is about to be
50 * rendered, allowing them to compute the final screen coordinates and dimensions correctly.
51 *
52 * @see helios::rendering::RenderTarget
53 * @see helios::scene::CameraSceneNode
54 * @see https://registry.khronos.org/OpenGL-Refpages/gl4/html/glViewport.xhtml
55 */
