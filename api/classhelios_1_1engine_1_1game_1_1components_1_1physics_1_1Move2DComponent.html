Detailed Description

Component for 2D physics-based movement with rotation.

Stores configuration and runtime state for 2D physics-based movement. This component is used by the movement system to apply smooth acceleration, deceleration, and rotation to the entity.

The component receives input via move() which sets target direction and throttle. The actual physics simulation (integration of velocity, application of dampening) is performed by the Movement2DSystem.
