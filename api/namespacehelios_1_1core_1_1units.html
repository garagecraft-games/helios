enum class helios::core::units::Unit
exportstrong

Enumeration of length and time units supported by helios.

Enumerator
Meter 

Base length unit (meters) in helios. 1.0f represents 1 meter.

Centimeter 

Centimeter helper unit for authoring convenience. Converted to meters (0.01f) when used.

Seconds 

Base time unit (seconds) in helios. 1.0f represents 1 second.

MilliSeconds 

Millisecond helper unit derived from seconds. Converted to seconds (0.001f) when used.
