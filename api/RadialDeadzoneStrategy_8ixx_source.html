14 /**
15 * @brief Implements a radial (circular) deadzone strategy for analog stick normalization.
16 *
17 * This strategy treats the deadzone as a circle centered at the origin. Input vectors
18 * with a magnitude less than or equal to the deadzone threshold are zeroed out completely.
19 * Input vectors exceeding the deadzone are rescaled so that the effective input range
20 * starts at the edge of the deadzone circle, providing a smooth and consistent response.
21 *
22 * The radial approach is generally preferred over axial deadzones because it eliminates
23 * diagonal movement artifacts and provides uniform sensitivity in all directions.
24 *
25 * **Algorithm:**
26 * 1. Calculate the magnitude (length) of the input vector.
27 * 2. If magnitude <= deadzone, set both axes to zero.
28 * 3. If magnitude > 1.0, normalize the vector to unit length.
29 * 4. Rescale the magnitude from [deadzone, 1.0] to [0.0, 1.0].
30 * 5. Apply the rescaled magnitude to the normalized direction vector.
31 *
32 * @see DeadzoneStrategy for the abstract interface.
33 */
