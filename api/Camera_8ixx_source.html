15 /**
16 * @brief Represents a camera for perspective or orthographic projection, deriving from SceneNode
17 * to allow for spatial positioning and transformation within the scene graph.
18 *
19 * The camera provides the view and projection matrix, whereas the view matrix
20 * is computed as the inverse of the camera's world transform.
21 *
22 * A Camera behaves just like a (non-renderable) SceneNode, except for the fact that
23 * it does not accept child nodes.
24 *
25 * An instance of `Camera` is added as a direct descendant of the scene's root node
26 * for free transform. It can be a child of a model hierarchy to inherit the
27 * corresponding positioning in the world.
28 *
29 * @todo implement class. Make sure localTransforms mark this class as dirty,
30 * for proper recalculating projection-/viewMatrix if needed.
31 * @todo check whether nodes should further be divided into Parent- and LeafNodes.
32 * Cameras would be LeafNodes then, not allowing to have child nodes.
33 */
