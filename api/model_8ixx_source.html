26 * @brief Creates an affine transformation matrix M' = M * T by "baking" the translation part into the model
27 * matrix.
28 * Any operation M' * v will make sure that v is translated in local space, then transformed into

30 *
31 * @param model
32 * @param v
33 * @return

38 t(1, 3) += t(1,0) * v[0] + t(1, 1) * v[1] + t(1, 2) * v[2];
