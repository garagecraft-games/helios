Detailed Description

Representative of a configurable Renderable rendered by the underlying GL API.

A Renderable is an aggregate consisting of Material and a Mesh, providing geometric primitives and material information.

Todo
a Renderable should be configured with optional, selected overrides for the associated mesh/material, so that for the used shared mesh/material objects, individual draw configs are possible.
