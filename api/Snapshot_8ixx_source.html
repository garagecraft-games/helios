19 /**
20 * @brief Represents an immutable snapshot of a SceneGraph at a given point in time.
21 *
22 * The class encapsulates all necessary data to render a single frame. It is
23 * designed to be a lightweight DTO to pass information to the rendering
24 * engine abstraction layer, such as the `RenderPass` class.
25 *
26 * A snapshot includes a camera's projection and view matrix that should be used
27 * for rendering the scene. A collection of `SnapshotItems` holds the information
28 * about the renderable objects in the scene, preferably computed by using a
29 * `FrustumCullingStrategy` (@see Scene::createSnapshot).
30 *
31 * The class is not copyable to enforce its immutability and prevent unintended
32 * duplication of potentially large data sets.
33 */
