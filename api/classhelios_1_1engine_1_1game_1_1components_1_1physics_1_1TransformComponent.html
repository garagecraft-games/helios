Detailed Description

Component that holds transformation data (position, rotation, scale).

The TransformComponent manages the spatial state of a GameObject. It stores the local transform (relative to parent) and the computed world transform. Changes to local properties mark the component as dirty, signaling systems to recompute the world transform.
