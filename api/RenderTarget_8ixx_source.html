17 /**
18 * @brief Represents a destination for rendering operations, such as a window's framebuffer.
19 *
20 * A RenderTarget encapsulates a buffer (e.g., the default framebuffer or an off-screen texture)
21 * into which a scene can be rendered. It manages one or more `Viewport` objects, which define
22 * specific rectangular areas within the target.
23 *
24 * When the RenderTarget is resized, it notifies all its associated viewports so they can update
25 * their states accordingly (e.g., camera aspect ratio).
26 *
27 * @todo This implementation currently represents only the **default framebuffer**. Future extensions
28 * should allow it to represent arbitrary framebuffer objects (FBOs), using indices.
29 * @see glGenFramebuffers
30 */
