Detailed Description

Command that triggers a shooting action on a GameObject.

This command encapsulates a shoot request with an intensity factor derived from trigger pressure or button state. When executed, it invokes ShootComponent::shoot() to initiate projectile spawning.

Note
The target GameObject must have a ShootComponent attached for this command to have any effect.
