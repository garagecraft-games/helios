Detailed Description

DTO for storing rendering-specific command information to be passed to the RenderQueue.

A `RenderCommand` does not extend the lifetime of the `renderPrototype_` as it's referenced by a weak pointer. If the referenced RenderPrototype is no longer valid when this command is about to be executed, this `RenderCommand` should be ignored for rendering.
