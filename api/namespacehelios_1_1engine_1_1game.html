Detailed Description

Game logic infrastructure using Command pattern for decoupled, deterministic game systems.

This namespace provides game logic infrastructure using the Command pattern for decoupled, testable, and deterministic game systems. It includes entity management (GameWorld, GameObject), input processing (InputSnapshot, InputHandler), and command execution (Command, CommandBuffer) facilities that enable advanced features like replay systems and undo/redo.
