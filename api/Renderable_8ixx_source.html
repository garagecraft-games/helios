24 /**
25 * @brief Representative of a configurable Renderable that references an immutable RenderPrototype
26 * and instance specific material property overrides.
27 *
28 * A Renderable aggregates a reference to an immutable RenderPrototype (the shared asset definition)
29 * with optional instance-specific material property overrides. This separation enables efficient
30 * batching of shared RenderPrototypes while allowing individual visual adjustments per Renderable instance.
31 *
32 * The Renderable's interface is API-agnostic, enabling efficient batching and per-instance customization,
33 * while abstracting the underlying API-specific rendering resources bundled within the RenderPrototype.
34 *
35 * Renderables are designed to be movable (e.g., in render queues) but not copyable to ensure unique
36 * instance identity in processing.
37 */
