Detailed Description

Transformation utilities for cameras, models, and scene graph inheritance.

This module provides transformation functions and utilities including:

  • **Camera transforms**: View matrix construction (`lookAt`)
  • **Model transforms**: Rotation, scaling, translation utilities
  • **TransformType**: Bitmask enum for selective transform inheritance
Note
Matrix operations `decompose()` and `transpose()` are available as member functions of `mat4` in `helios.math.types`.
